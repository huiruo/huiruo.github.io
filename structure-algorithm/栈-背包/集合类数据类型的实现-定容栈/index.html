集合类数据类型的实现-定容栈

1.不通用，表示容量固定的字符串栈的抽象数据类型​

只能处理String 值，要求指定一个容量且不支持迭代。
FixedCapacityStackOfStings
/*
* 优点:
* 1.栈的乞丐版本
* 2.push和pop和栈的长度无关
缺点:
* 1.不支持各种类型的(泛型);
* 2.数组大小被写死了.没办法动态的改变
* 3.对象游离了没有被回收
* 4.不支持迭代
*
* 改进版本:算法1.1 下压栈 能够动态调整数组大小
*  */
public class FixedCapacityStackOfStrings {
    private String[] a;
    private int n = 0;

    // 创建 固定容量空栈
    public FixedCapacityStackOfStrings(int capacity) {
        a = new String[capacity];
    }

    // 添加一个字符串
    public void push(String item) {
        //a[n++]=item
        a[n] = item;
        n++;
    }

    // 删除最近添加的字符串
    public String pop() {
        //return a[--m]
        n = n - 1;
        return a[n];
    }

    // 栈是否空
    public boolean isEmpty() {
        return n == 0;
    }

    // 栈的字符串数量
    public int size() {
        return n;
    }
}

使用泛型实现栈​

FixedCapacityStack<Item>

有一个细节非常重要：
希望用一下代码在 FixedCapacityStack 的构造函数的实现中创建一个泛型的数组：
a = new Item[cap];

但是java 创建泛型数组是不允许的，这里要用到类型转换：

a = (Item[]) new Object[cap];

调整数组的大小​

为避免移除，push() 需要检测代码中栈是否满，动态数组的大小，又不至于过大。
实现一个方法将栈移动到另一个大小不同的数组中：

private void resize(int max)
{
    // 将大小为N<= max 的栈移动到一个新的大小为max 的数组中
    Item[] temp=(Item[]) new Object[max];
    for(int i=0;i<N;i++)
        temp[i]=a[i];
    a=temp;
}

现在，push 会检查栈大于N和数组大小 a.length 是否相等 决定是否扩容。
如果没有多余，将数组长度加倍，然后就可以和从前一样用 a[N++] = item 插入新元素。

public Item pop()
{
    Item item= a[--N];
    a[N]=null;
    if(N>0 && N==a.length/4)  resize(a.length/2);
    return item;
}

类似，pop 将数组减半：
public Item pop()
{
    Item item= a[--N];
    a[N]=null;
    if(N>0 && N==a.length/4)  resize(a.length/2);
    return item;
}

总结：这样就实现了栈不会溢出，使用率也永远不会低于四分之1
